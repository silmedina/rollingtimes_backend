# Rolling Times

## Descripcion

Este es el proyecto final de RollingCode School Fullstack integrando front-end con React y Backend con NodeJS. 


## Instalacion 

Para instalar el proyecto ejecutar con npm

```bash 
  npm install
```

## Ejecucion 

Para correr el proyecto ejecutar con npm:

```bash 
  npm run dev
```
